# bun-ts-starter

A TypeScript library

## Installation

```bash
npm install bun-ts-starter
```

## Usage

```typescript
import { greet } from 'bun-ts-starter'

console.log(greet('World')) // Hello, World!
```

## Development

```bash
# Install dependencies
bun install

# Build
bun run build

# Develop with watch mode
bun run dev

# Run tests
bun run test
```
